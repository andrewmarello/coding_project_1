# Coding Project 1

Coding Project 1 - User Feedback Form: Capturing and Responding to User Input - Andrew Marello 

Creating an interactive and responsive feedback form to gain consumer insights.
Technologies used include Visual Studio Code and GitHub.
The project runs by entering name, email, and any potential comments that one might have for the business. After clicking submit, the form will clear and the response will be logged beneath the form entry area.
After building this interactive form, I learned about how to add some cool features, such as hover buttons and creating error messages so that ir can prompt a user to make sure fields are filled out properly.
